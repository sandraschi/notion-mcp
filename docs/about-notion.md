# About Notion

## What is Notion?

Notion is a workspace platform that combines notes, documents, databases, wikis, and project management into one tool. It was created by Notion Labs Inc., founded in 2013 by Ivan Zhao and Simon Last.

Unlike traditional note-taking apps that are file-based (markdown files, plain text), Notion uses a block-based editor where everything is a "block" — text, images, tables, code, embeds — that can be nested, reordered, and linked. Pages can be databases, and databases can have custom properties (select, date, relation, formula, etc.).

## History

- **2013** — Notion Labs founded in San Francisco by Ivan Zhao and Simon Last
- **2015** — First public release. Initially an all-in-one document + database tool
- **2018** — Notion 2.0 launched with a redesigned editor, templates, and API
- **2019** — Introduced "Notion for Teams" with collaboration features
- **2020** — Explosive growth during remote work. Reached 4M+ users
- **2021** — Series C funding at $10B valuation (20M+ users). Public API launched
- **2022** — Notion AI announced. Acquired Flowdash, Automate.io
- **2023** — Notion AI general availability. Notion Calendar launched
- **2024** — Notion Sites (publish pages as websites). API v2 preview
- **2025** — Connected databases, improved automations, deeper API capabilities
- **2026** — Continued expansion of AI features, API version 2026-03-11

## Usage Stats

Sources: Notion official blog, public investor reports, web traffic estimates.

- **Users**: 100M+ (2025 estimate, up from 20M in 2021)
- **Team workspaces**: Millions of teams across companies of all sizes
- **Platforms**: Web, macOS, Windows, iOS, Android
- **Integrations**: 100+ through public API and Zapier/Make connectors
- **Block count**: Notion stores and indexes billions of blocks across all workspaces

From a technical perspective, Notion's API supports:
- 3 requests per second per integration (rate limit)
- 100 results per page (capped at 100 for most list endpoints)
- Block depth up to 200 levels (practical limit is lower for performance)
- File attachments up to 5MB via the API
